# Code-Alpha-Basic-ChatBot
A simple rule-based chatbot built in Python using loops and if-else conditions to respond to user inputs.



Basic Chatbot 🤖

A simple console-based chatbot built using Python that responds to predefined user inputs.

---

📌 Concepts Used

- Control Statements ("if-elif-else")
- Loops ("while")
- User Input Handling ("input()" function)

---

⚙️ Features

- Responds to basic inputs like:
  - "hello"
  - "how are you"
  - "bye"
- Runs continuously until user exits
- Simple and easy-to-understand logic

---

▶️ How to Run

1. Install Python
2. Run the file:

python chatbot.py

---

💬 Example Output

You: hello
Bot: Hi!

You: how are you
Bot: I'm fine, thanks!

You: bye
Bot: Goodbye!

---

⚠️ Limitations

- Only works with fixed input phrases
- Case-sensitive (e.g., "Hello" won’t work)
- No AI or natural language processing

---

🔥 Future Improvements

- Add more responses
